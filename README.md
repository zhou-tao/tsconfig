# TS Config Base File

For unified management typescript config in every app.

### Install

```js
npm i @tooz/tsconfig
```

### Usage

add `extends: '@tooz/tsconfig/tsconfig.app.json'` in your `tsconfig.json`.

```json
{
  "extends": "@tooz/tsconfig/tsconfig.app.json",
  "include": "",
  ...
}
```
