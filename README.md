# TS Config

:heart: Typescript applications common configuration files.

### Install

```js
npm i @tooz/tsconfig -D
```

### Usage

Add to the extends field in your tsconfig.json.

- base

```json
{
  "extends": "@tooz/tsconfig",
}
```

- web

```json
{
  "extends": "@tooz/tsconfig/web"
}
```

...
