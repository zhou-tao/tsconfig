# TS Config Base File

For unified management typescript config in every app.

### Install

```js
npm i @tooz/tsconfig -D
```

### Usage

add `extends: '@tooz/tsconfig/tsconfig.vue.json'` in your `tsconfig.json`.

```json
{
  "extends": "@tooz/tsconfig/tsconfig.vue.json",
  ...
}
```
